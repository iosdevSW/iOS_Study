# STT (Speech to Text)
 사람이 말하는 음성 언어를 컴퓨터가 해석해 그 내용을 문자 데이터로 전환하는 처리
 Open API를 찾을 수 없어 고민중에 Apple에서 제공하는 프레임워크를 찾아 스터디하여 구현하였다.

## 사용 프레임워크 
- AVFoundation
- SPEECH
